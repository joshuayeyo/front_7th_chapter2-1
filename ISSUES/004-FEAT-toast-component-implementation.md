# 004 - FEAT - Toast 컴포넌트 구현

## Description

동적 Toast 알림 컴포넌트를 구현하여 사용자에게 피드백을 제공합니다. 기존 정적 템플릿을 클래스 기반 컴포넌트로 전환하여 재사용성과 유지보수성을 향상시킵니다.
